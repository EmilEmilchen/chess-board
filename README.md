# Digital RFID-reading Chess Board

BOM files from suppliers are in their respective directories, DigiKey is strongly preferred, LCSC part numbers of equal or equivalent parts are given in the KiCad Symbol Table in case of using JLCPCB assembly services later.
Here is a [link to the BOM on the DigiKey website](https://www.digikey.de/de/mylists/list/6JU4LPDGX9).
Note that 4 (2x2) connectors are to be ordered from amazon or eBay since thats not only cheaper but also simpler (these are still referenced in the DigiKey BOM but not connected to any DigiKey part, same goes for the MFRC522 RFID main IC, that's about 5-10x cheaper when ordered from AliExpress (around 0.47€/pcs).
The materials in the BOM are for one sub-board (2x2 matrix of RFID readers) with a slight excess in some cases where it was cheaper to order e.g. 10 pcs instead of 8 due to bulk discounts.
